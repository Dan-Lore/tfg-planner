# Глоссарий

## Термины проекта

| Термин | Определение |
|--------|-------------|
| **Мнемосхема** | Визуальная схема производственной линии на холсте |
| **Узел (Node)** | Экземпляр механизма на схеме |
| **Связь (Edge)** | Направленное соединение потока между узлами |
| **Pack data** | Нормализованный JSON-бандл рецептов и машин для одной версии модпака |
| **`.tfgp`** | Файл сохранённой схемы (TFG Planner Graph) |
| **Масштабирование A** | Ручное копирование участка линии на холсте |
| **Масштабирование B** | «Умножить выходы» — дробный множитель; `machineCount` округляется; узлы не дублируются |
| **machineCount** | Целое число машин, которое представляет узел на схеме (≥ 1) |
| **Идеальное machineCount** | Дробное значение до округления (внутри калькулятора) |
| **Округление машин** | `ceil(ideal)` → min 1 → полный пересчёт схемы |
| **Масштабирование C** | Целевая скорость продукта на выходе линии → пересчёт потоков |
| **Группа** | Выделенный набор узлов (для режима B и организации схемы) |
| **Undo / Redo** | Отмена и повтор действий: Ctrl+Z / Ctrl+Y |

## Термины TerraFirmaGreg / GregTech (кратко)

| Термин | Определение |
|--------|-------------|
| **TFC** | TerraFirmaCraft — ранняя стадия выживания, металлургия |
| **GT / GTCEu** | GregTech Community Edition Unofficial — поздняя автоматизация |
| **EBF** | Electric Blast Furnace — электропечь |
| **MV / HV / EV …** | Voltage tiers — уровни энергии |
| **Overclock** | Ускорение машины повышением напряжения (со штрафом к энергии) |
| **Parallel** | Параллельная обработка нескольких партий в одной машине |
| **KubeJS** | Скрипты модпака, часто определяют или меняют рецепты |
